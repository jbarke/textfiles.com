PLANET EARTH NETWORK NODEMAP 09/94 PEN supports three HUB levels below the host. This allows a wide coverage distribution system with the trade off of some additional delay in message delivery. With the levels of HOST, SUPER REGIONAL, REGIONAL, HUB & NODE, a message entered at the node level would generally take 9 days to be delivered to the entire network. (assuming daily mail transfers between levels) This is the extreme case; but, illustrates how this can negatively influence conversation. Closer systems could have made several replies before more distant systems even saw the first remark. With host change from Intelec Online to West Los Angeles BBS (WLA), the network actually has grown yet another level. Reassignment of HUB status is necessary. This is will not change any present connections; but, it will change how the network grows. HOST - central point of network SuperRegional - connects directly to Host Regional Hub - connects to S/R or Host Hub - connects to Regional, S/R or Host Node - connects to any of the above, the only level that is not required to carry all conferences.
